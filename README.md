# meu_app

Aplicativo de nutrição desenvolvido com Dart + Flutter.

## Funcionalidades iniciais

- Resumo diário de calorias e água
- Metas nutricionais (proteínas, carboidratos e gorduras)
- Plano de refeições do dia

## Como executar

1. Instale o Flutter SDK.
2. Neste diretório, execute:

```bash
flutter pub get
flutter run
```

## Estrutura principal

- `lib/main.dart`: ponto de entrada e interface inicial do app.
